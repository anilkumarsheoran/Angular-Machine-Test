Introduction

The purpose of this machine test is to assess the candidate's knowledge in front-end development with the technologies like AngularJS, Javascript, CSS, and HTML. We recommend the candidate to make the best effort for solving the given problem.

Problem Description:-

Create an angular application which calls a REST API using a service or a factory. You also need to implement routing for different views which we need for showing data to the users and a form from where they can submit new data. You can also create a directive for doing validation of form data. You can also use some styles to beautify your app. Also keep in mind the application structure and the folder structure.

Requirements:

1. Create an angular module.
2. Create atleast one controller.
3. Create atleast one service/factory.
4. Create atleast one directive.
5. Create routes.
6. Use $httpService.
7. Create atleast one GET method
8. Create atleast one POST method.

API Structure:

Url of the API:- http://jsonplaceholder.typicode.com/
There are various Posts which are saved on the API.
GET:- http://jsonplaceholder.typicode.com/posts/
Returns Posts array.
POST:- http://jsonplaceholder.typicode.com/posts/
Add a new post. Returns the object when Post succeeds. This call don't add object into posts array.
